# 🌕 月之符文占卜系統（Moon Runes / Lunar Codex）PWA

由 **月語之境工作室（EsotericVerse）** 製作的月之符文占卜系統。  
本專案為一套可部署於 GitHub Pages 的漸進式網頁應用程式（PWA），  
同時也是「月之符文（Moon Runes / Lunar Codex）」語義系統的實作載體之一。

---

## ✨ 主要特色

- 🎯 **多種占卜模式**：支援不同占卜流程與結果呈現方式  
- 🌙 **完整月相系統**：結合真實月相與符文固定月相  
- 🧩 **64 張符文語義庫**：每張符文皆具備固定語義、分組與月相  
- 🤖 **AI 輔助解讀**：以語法與結構為基礎的輔助生成（非隨機生成）  
- 📱 **PWA 支援**：可離線使用、可安裝至桌面  
- 🎨 **響應式設計**：支援桌機與行動裝置

## 🌙 月之符文系統（Moon Runes / Lunar Codex）

「月之符文」並非單純卡牌占卜，而是一套**具備語義結構與生成規則的符文系統**。

### 系統核心概念

- **64 張核心符文（1–64）**
  - 每張符文皆具備：
    - 固定語義主題
    - 固定月相（新月／上弦／滿月／下弦）
    - 所屬符文分組（靈魂、連結、生命、自然、礦物、元素、時序、亂序）
- **四向能量狀態**
  - 正位（↑）／半正位（＋）／半逆位（—）／逆位（↓）
  - 方向僅影響「能量狀態」，不改變符文本體語義
- **命運語法（Destiny Syntax）**
  - 符文 × 月相 × 方向可組合為結構化語句
  - 支援單卡、雙卡、三卡與多卡占卜模型

> 本倉庫所實作之占卜流程，皆遵循上述語義與語法規則。

### 個人擴展層（說明性標註）

- **65：命**
- **66：玄**
- **0：無**

上述符文屬於作者個人系統層，不影響 1–64 核心符文的公共一致性，  
僅於特定占卜規則或個人研究中使用。

## 🚀 功能詳解

### 🎯 占卜系統
### 🔮 占卜模式一覽（Index 入口）

本系統於首頁（`index.html`）提供多種占卜入口，可依使用需求選擇不同抽牌結構與占卜深度：

- **單卡占卜**
  - 基礎占卜模式  
  - 適合快速詢問、即時狀態觀測  
  - 抽取 1 張符文，包含 4 種方向能量狀態

- **每日占卜**
  - 專為日常使用設計  
  - 結合當日真實月相進行解讀  
  - 可由首頁或結果頁直接進入

- **雙卡占卜**
  - 用於觀察兩個主題之間的互動與張力  
  - 適合關係、抉擇與內外衝突分析

- **三卡占卜**
  - 對應「源 → 轉 → 合」的命運語法結構  
  - 常用於事件發展、過程解析與結果趨勢判讀

- **五卡占卜**
  - 提供更高複雜度與完整度的占卜結構
  - 適合長期議題、階段性命運評估

### 🧭 OW3gs 方式

OW3gs 為作者設計的「三向導引式抽牌流程」，用於快速排列出「源 → 轉 → 合」的命運語法：

- **O（Origin）**：釐清提問者的核心議題與背景設定。
- **W（Weave）**：在系統中抽取三張符文，依序對應「起源／推進／整合」的語法節點。
- **3gs（Three Guided Steps）**：以三步驟完成解讀——記錄符文與方向、比對固定月相、依照命運語法撰寫結論。

實務操作建議：

1. 於 `index.html` 選擇三卡占卜或相關入口，保持抽牌順序一致（第一張為 O，第二張為 W，第三張為收束）。
2. 在結果頁 (`result.html` / `fate.html`) 依序檢視月相互動，確認每張符文的固定月相與當日月相關係。
3. 依循上述三步驟撰寫解讀，避免自由生成或偏離符文既有語義，確保結論可回溯至符文資料庫。

> OW3gs 旨在提供結構化、可重現的占卜流程，方便研究者與使用者以相同步驟重演結果。

此外，首頁亦提供以下輔助入口：

- **符文資料查詢**
  - 瀏覽各符文的語義、分組與固定月相屬性

- **命運句語法圖鑑**
  - 系統語法與結構說明文件（Word 格式）

- **系統介紹文章**
  - 月之符文設計理念與系統背景說明（外部連結）

> 所有占卜模式皆基於同一套符文語義與月相規則，  
> 差異僅在於抽牌數量與解讀結構層級。


## 🤖 AI 輔助解讀說明

本專案中的 AI 並非自由生成文字，而是：

- 依照「符文語義 × 月相 × 方向」結構
- 輔助生成**語法一致、語義受限**的解讀文字
- 可視為「命運語法的解讀引擎」

目前相關功能仍屬測試與實驗階段。
